# Logo Placeholder

Este diretório deve conter o logo da empresa (logo.png).

Para adicionar o logo:
1. Crie uma imagem PNG com dimensões recomendadas de 200x200px
2. Salve como `logo.png` neste diretório
3. O logo será exibido no cabeçalho do site

Formato recomendado:
- Tipo: PNG com transparência
- Dimensões: 200x200px ou 300x300px
- Tamanho: < 100KB
